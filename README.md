Unhealthy
---------

Unhealthy is Custom AWS Personal Health Dashboard.

## Installation

```sh
git clone https://github.com/winebarrel/unhealthy.git
cd unhealthy
bundle install
```

## Getting Started

```sh
bundle exec ruby app.rb
```

```sh
open http://localhost:4567
```
